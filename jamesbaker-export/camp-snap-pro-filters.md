I got a [Camp Snap Pro](tab:https://www.campsnapphoto.com/products/cs-pro-camera) for Christmas. One of the main reasons I wanted it was to spend less time at my computer editing photos.

Camp Snap lets you build your own [filters](tab:https://www.campsnapphoto.com/pages/filters) and load them onto your camera.

After playing around on the website and building my own filter, I downloaded a file called `std.flt`. On a whim, I tried opening it in a text editor:

```
lum:5
contrast:3
rgain:0
ggain:-9
bgain:-5
hue:0
sat:10
```

_Nice!_

Each slider in the web UI is saved as a `key:value` pair. Now I can experiment much faster. Plus I can use [git](tab:https://git-scm.com/) to version control my `.flt` files.

I took the camera to a [Marmozets](tab:https://marmozets.band/) gig. Because it’s a bit cheaper, I‘m not as worried about it getting broken.
